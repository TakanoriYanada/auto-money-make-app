# AI比較ナビ

AIツール比較メディアのNext.jsアプリ

## 🚀 開発・実行

```bash
# 開発サーバー起動
npx vercel dev
```

## 🌐 デプロイ

```bash
# 本番デプロイ
npx vercel --prod
```

## 🔧 環境変数

```bash
# 環境変数を設定
npx vercel env add <VARIABLE_NAME> production
```

## 📄 収益化設定

1. docs/REVENUE_MODEL.md を参照
2. 各ASPでアフィリエイト登録
3. 環境変数にIDを設定
4. Vercelでデプロイ
